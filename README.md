# DISTRIAI - Distributed AI Compute Network

Landing page for DISTRIAI - Enterprise-ready infrastructure for decentralized AI inference at scale.

## Deploy on Vercel

1. Push this repository to GitHub
2. Import the repository on [Vercel](https://vercel.com)
3. Vercel will automatically detect it as a static site
4. Deploy!

## Files

- `index.html` - Main landing page
- `logo.png` - DISTRIAI logo
- `trees-bg.jpg` - Background image (pixelated trees)
